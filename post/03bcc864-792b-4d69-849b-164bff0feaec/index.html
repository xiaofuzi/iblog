Getting Started

This page will help you install and build your first React Canvas App. If you already have React Canvas installed, you can skip ahead to the Tutorial.

Quick Start

Assuming that you have NodeJS installed, you can use npm to install the create-canvas-app CLI command line utility:

npm install -g create-canvas-app

The run the following commands to create a new React Canvas project called "AwesomeProject":

create-canvas-app AwesomeProject
cd AwesomeProject
npm run start

This will start a develoment server for you.

Congratulations! You've successfully run your first React Canvas App.
