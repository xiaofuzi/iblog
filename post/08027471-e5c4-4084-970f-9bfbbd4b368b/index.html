Canvas

The root react component for react canvas App.

In the following example will render "Hello world!" String on the screen.

render(
  <Canvas width={750} height={1334} display="flex">
    <View >
        <Text>Hello world!</Text>
    </View>
  </Canvas>,
  document.getElementById("root").appendChild($canvas)
);
