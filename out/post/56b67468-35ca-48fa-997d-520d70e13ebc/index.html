Learn the basics

React Canvas is like React, but it uses canvas components instead of web dom components as build blocks. So to understand the basic structure of a React Canvas App,you need to understand some of the basic React concepts, like JSX, components, state, and props. if you already know React, you still need to learn some React-Canvas-specific stuff, like the native components. This tutorial is aimed at all audiences, whether you have React experience or not.
