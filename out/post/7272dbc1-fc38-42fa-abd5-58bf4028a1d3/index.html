介绍

Voit.js 是什么？

Voit.js 是一套基于 canvas 的用于构建用户界面的渐进式框架。Voit 被设计为可以自底向上逐层应用，易于上手，同时可以便捷的与第三方库结合使用。

起步

如果你已经了解 vue/react 这样的前端框架，那么你将会非常容易的理解本框架做了什么以及快速的学习使用。

声明式渲染

Voit.js 采用的是类 jsx 的模板语法来声明式地将数据渲染为 canvas 元素。

<View>
  <Text>{{ message }}</Text>
</View>
const app = new Voit({
  el: '#el',
  data: {
    message: 'Hello world!'
  }
})

我们已经成功创建了第一个 Voit.js 应用。在界面上会渲染出 Hello world! 文本，在这个过程中，Voit 会创建一个 canvas 树并将数据和 canvas 节点关联起来，所有的东西都是响应式的。

除了文本插值的语法，我们还可以通过属性的方式来绑定数据：

<View tw='750' th='1334'>
  <Text text='message'></Text>
</View>

这里的 tw, th, text 我们称之为属性，它们会在渲染 canvas 元素的时候将数据响应式的关联到元素本身。

条件与循环

控制切换一个元素是否显示：

<Canvas id="app-3">
  <text v-if="seen">现在你看到我了</text>
</Canvas>
let app3 = new Voit({
  el: '#app-3',
  data: {
    seen: true
  }
})

继续在控制台输入 app3.seen = false, 你会发现之前显示的消息消失了。

这个例子演示了我们不仅可以把数据绑定到 canvas 元素上，还可以作用于 canvas 树结构本身。

还有其它很多的指令，每个都有特殊的功能。例如，v-for 指令可以绑定数组的数据来渲染一个列表：

<Canvas id='app-4'>
  <Text v-for='todo in todos'>
    {{ todo.text }}
  </Text>
</Canvas>
let app4 = new Voit({
  el: '#app-4',
  data: {
    todos: [
      { text: '学习 JavaScript' },
      { text: '学习 Vue' },
      { text: '整个牛项目' }
    ]
  }
})

在控制台里，输入 app4.todos.push({ text: '新项目' })，你会发现列表最后添加了一个新项目。
